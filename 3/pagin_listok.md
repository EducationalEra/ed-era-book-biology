
# Пагiн: листок

<div class="eoz-wrap">
<span class="eoz">Визначення</span>
<div class="eoz-text">
<b>Листок</b> — це бiчна частина пагона, якiй властивий обмежений рiст.
</div>
</div>

Листок виконує такi функцiї:
* Фотосинтез
* Газообмiн (дихання)
* Транспiрацiя (випаровування) 
* Вегетативне розмноження.
